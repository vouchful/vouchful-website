How do I get started using Email Kit?

There are a few common ways to get started. You can download the base layouts and then begin adding in elements and components using your favorite code editor. Or if there is a specific template you need, you can just download that and customize it. After adding in your content, just import the HTML into your Email Service Provider for testing. Before sending, make sure you inline the CSS.

How do I use Email Kit with my current Email Service Provider (ESP)?

Because nearly every major ESP allows for HTML to be imported, Email Kit can be integrated into your existing workflow. After downloading the templates, edit them using your code editor of choice and import them directly into your ESP. Before sending, make sure you inline the CSS and add in any custom tags required by your ESP.

Does Email Kit work on mobile, tablet and desktop?

With email usage distributed across mobile, tablet and desktop, Email Kit has been developed to work across all three including popular devices such as iPhone, iPad and Pixel as well as desktop clients such as Apple Mail and Outlook.
