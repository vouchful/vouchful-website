Usage

With traditional web development, there is generally a clear distinction between buttons and links.

  • Use a link when you want to navigate to another page.
  • Use a button when you want to take an action on a page.

However, when developing HTML emails, these two elements begin to blur. For example, forms are not supported in an email, but you might want to use a button for your call-to-action. In those cases, instead of using a button tag, you will instead use Email Kitâ€™s built-in classes to style an a tag to look like a button.
